Quick Start

1. Get Your API Key

Sign up for a free account to receive your API key instantly. No credit card required for the free tier.

2. Make Your First Request

curl -X POST https://multi-evm-gateway-197221342816.us-central1.run.app/ethereum \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "eth_getBalance",
    "params": ["0x742d35cc6634c0532925a3b8d3ac74e4dc3c8ef5", "latest"],
    "id": 1
  }'

3. Handle the Response

{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x1b1ae4d6e2ef500000"
}

WebSocket Connections

Real-time Data Streaming

Connect to our WebSocket endpoints for real-time blockchain data:

const ws = new WebSocket('wss://api.aetherweb3.xyz/ws/v1/ethereum?auth=YOUR_API_KEY');

ws.onopen = function() {
  // Subscribe to new blocks
  ws.send(JSON.stringify({
    "jsonrpc": "2.0",
    "method": "eth_subscribe",
    "params": ["newHeads"],
    "id": 1
  }));
};

ws.onmessage = function(event) {
  const data = JSON.parse(event.data);
  console.log('New block:', data);
};
